Constraint-Based (Design Once)

Recommendation: Hybrid B + C

Design primarily on desktop using Approach C (constraint-based layout rules). The canvas generates clean CSS Grid/Flex. Switch to Tablet/Mobile preview to see auto-adapted output. Override specific elements per breakpoint only where needed — not rebuilding the whole layout.

For the designer

When placing elements you set layout behavior — "stretch full width", "stack on mobile", "maintain aspect ratio" — not just X/Y coordinates. One extra property per element, saves hours of CSS debugging.

For the output

The system generates real CSS Grid + Flexbox with proper breakpoints. Volt elements are SVG so they scale perfectly. The designer canvas IS the source of truth — no gap between designer and browser.
